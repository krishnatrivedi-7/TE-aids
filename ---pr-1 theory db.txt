### **8ï¸âƒ£ SET OPERATORS**

These combine results from two or more queries.

1. **UNION**

   ```sql
   SELECT first_name, city FROM Employees
   UNION
   SELECT first_name, city FROM Old_Employees;
   ```

   * Combines results of both tables and **removes duplicates**.

2. **INTERSECT (Simulated via INNER JOIN)**

   ```sql
   SELECT e.first_name, e.city FROM Employees e
   INNER JOIN Old_Employees o ON e.first_name = o.first_name;
   ```

   * Shows only names that appear in **both** tables.

3. **MINUS (Simulated via LEFT JOIN and IS NULL)**

   ```sql
   SELECT e.first_name, e.city FROM Employees e
   LEFT JOIN Old_Employees o ON e.first_name = o.first_name
   WHERE o.first_name IS NULL;
   ```

   * Shows records present in **Employees** but **not in Old_Employees**.
